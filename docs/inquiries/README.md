# Farmer & Rep Inquiries Log

Each inquiry lives in its own dated file in this folder. Filename convention:

    YYYY-MM-DD_<rep-or-company>_<grower-or-topic>.md

Keep a short frontmatter block on each entry so the log is greppable:

```yaml
---
date: 2026-04-21
rep: Dusty
company: ReLineHybrids Seed Co
grower: (anonymized or named)
state: IA
crop: corn
acres: 320
topics: [lignin biostim, N rate cut, sap testing, feedlot manure]
---
```

Then the body: the question verbatim, soil/agronomy context, and the answer we gave.

## Why we keep this

1. **Pattern recognition** — if 5 reps ask the same question, that's a marketing/education gap
2. **Trial design feedback** — real field scenarios point us at what data our cooperator trials need to collect
3. **Follow-up** — we can check back in with reps/growers on how their season went
4. **Institutional memory** — answers compound; next time a similar question comes in, we can reference the prior response instead of rebuilding it
