# Bio Trial Farmer Video

Mobile-first Remotion reel built from `docs/farmer-explainer.md`.

Output:

- `out/farmer-explainer-reel.mp4`
- `out/farmer-explainer-poster.png`

Commands:

```bash
npm install
npm run dev
npm run render
npm run render:poster
```

Notes:

- Format is `1080 x 1920` for Instagram Reels / TikTok / Facebook vertical video.
- The edit leans on honesty, ease, and public proof instead of making agronomy promises.
- Existing repo visuals are reused from the landing page and product image.
